This project is a simple implementation of a CHIP-8 emulator written in pure C.  
CHIP-8 is a virtual machine from the 1970s, often used as a starting point for emulator development.

## ⚙️ Features

- CHIP-8 CPU implementation
- Memory management (4KB)
- Opcode decoding and execution
- Basic input handling
- Graphics rendering (64x32 display)
